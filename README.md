# DimensionDoor Home Assistant Add-on Repository

Secure remote access to your Home Assistant instance without port forwarding or VPN.

## Add-ons

### [DimensionDoor Remote Access](./dimensiondoor)

Connects your Home Assistant to [DimensionDoor](https://dimensiondoor.cloud) for secure remote access via a unique `*.dimensiondoor.cloud` subdomain with SSL.

## Installation

Click the button below to add this repository to your Home Assistant:

[![Add to Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Fprototypeforge%2Fdimensiondoor-ha-addon)

Or manually:

1. Go to **Settings > Add-ons > Add-on Store**
2. Click the **three-dot menu** (top right) > **Repositories**
3. Add: `https://github.com/prototypeforge/dimensiondoor-ha-addon`
4. Find **DimensionDoor Remote Access** and click **Install**

## Quick Start

1. Create an account at [app.dimensiondoor.cloud](https://app.dimensiondoor.cloud)
2. Create a tunnel to get your auth token
3. Install the add-on and enter your token
4. Access your HA from anywhere!
